# Portfolio Client React

Frontend para mi portfolio personal. Consume una API REST externa y cuenta con sistema de enrutamiento y filtrado de proyectos.

## Características

- **Arquitectura:** Separación Lógica/Vista (Services Pattern).
- **Styling:** Tailwind CSS v3.
- **Routing:** React Router DOM v6.
- **API Consumption:** Axios con manejo de errores centralizado.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Axios

## Instalación

1. Clonar el repositorio.
2. `npm install`
3. `npm run dev`
